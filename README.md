# MC Status - Widget for Minecraft

This is a rather simple Swift 5 app to show you the current status of a Minecraft Server.<br>
Its main feature is the added Widget that lets you see who is online right from your homescreen/notification center!<br>
This app uses catalyst and is available for both iOS and macOS!<br><br>

Get it on the app store: https://itunes.apple.com/us/app/server-widget-for-minecraft/id1408215245?ls=1&mt=8<br><br>
